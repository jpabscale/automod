Setting up automod

  • In .config.json:

    "Projectlsa": {
  "aesKey": "",
  "contentPaks": "SpacePrototype/Content/Paks",
  "directory": "",
  "mapUri": "",
  "repakPackOptions": "",
  "unrealEngine": "4.26",
  "zen": false
}

  • In VSCode User settings.json:

    "Projectlsa.directory": "",
"Projectlsa.contentPaks": "SpacePrototype/Content/Paks",
"Projectlsa.unrealEngine": "4.26",
"Projectlsa.mapUri": "",
"Projectlsa.aesKey": "",
"Projectlsa.zen": "false",
"Projectlsa.repakPackOptions": "",

See the Prerequisites section on how to extract a game AES key.

Fill in the .directory; note that .directory uses the operating system path (<drive>:\<path-to-game>).
